# C7 - Core

[![Slack][slack-badge]][slack-url]

Core standards for Swift.

## Installation

- Add `C7` to your `Package.swift`

```swift
import PackageDescription

let package = Package(
    dependencies: [
        .Package(url: "https://github.com/open-swift/C7.git", majorVersion: 0, minor: 10)
    ]
)
```
